Task Tracker
Overview
Task Tracker is a simple web application that allows CURD operations for managing tasks.
The project consists of an ASP.NET Core Web API backend and a lightweight JavaScript frontend that communicates with the API.

Technologies Used
Backend
-	ASP.NET Core Web API
-	Entity Framework Core
-	InMemory Database
-	System.Text.Json
Frontend
-	HTML
-	JavaScript (ES Modules)
-	Fetch API
-	Boostrap intergrated css styling

Running the Project Locally: 
Clone The repo: https://github.com/YongaJ/TrackerTask/
Build and restore nuget packages if need be. 
Run the project. The API and Web are both set to run as startup apps on port 709 7(Web) and 7250 (API/Swagger).
Urls: https://localhost:7097/ and https://localhost:7250/swagger/index.html
